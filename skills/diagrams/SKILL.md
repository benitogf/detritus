# /diagrams — Mermaid Diagram Reference

For the full Mermaid syntax reference with examples of every diagram type, call `kb_get(name="plan/diagrams")` if the detritus MCP server is available.

## Quick Decision Table

| Diagram | When to Use |
|---------|-------------|
| **Flowchart** | Control flow, decision trees, process steps |
| **Sequence** | API calls, request/response flows, component interaction order |
| **ER** | Database schemas, data model relationships |
| **State** | Lifecycle states, FSM, connection states |
| **Class** | Interface hierarchies, struct relationships |
| **Gantt** | Timelines, project phases |
| **Pie** | Proportional breakdowns |
| **Mindmap** | Brainstorming, concept maps |
